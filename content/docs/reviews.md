# Система отзывов

В этом разделе подробно описана система отзывов клиентов, их создание, настройка и отображение на сайте.

## Обзор системы

Система отзывов представляет собой автоматизированный компонент, который:
- Собирает все отзывы из секции `content/reviews/`
- Отображает их на всех страницах сайта
- Поддерживает рейтинги, должности и порядок отображения
- Адаптируется под различные устройства

## Структура отзывов

### Расположение файлов
Все отзывы хранятся в директории `content/reviews/` в формате Markdown файлов.

### Автоматическое отображение
Отзывы автоматически отображаются на всех страницах благодаря подключению partial `reviews.html` в основных шаблонах.

## Создание отзыва

### 1. Создание файла
Создайте новый Markdown файл в `content/reviews/` с именем, отражающим автора отзыва (например, `john-doe.md`).

### 2. Фронтматтер (метаданные)
Каждый отзыв должен содержать фронтматтер с обязательными и опциональными полями:

```toml
+++
title = "Имя Фамилия"        # Обязательно - имя автора отзыва
date = "2025-02-01"          # Обязательно - дата отзыва
role = "Должность"            # Опционально - должность/роль автора
rating = 5                    # Опционально - рейтинг от 1 до 5
weight = 1                    # Опционально - порядок отображения
image = "/img/client.jpg"     # Опционально - фото клиента
+++
```

### 3. Содержание отзыва
После фронтматтера добавьте текст отзыва в формате Markdown:

```markdown
Текст отзыва клиента. Может содержать несколько абзацев и поддерживает все возможности Markdown.

Дополнительные детали, впечатления и рекомендации от работы с компанией.
```

## Поля отзыва

### Обязательные поля

#### `title`
- **Тип**: строка
- **Описание**: Имя автора отзыва
- **Пример**: `title = "Анна Петрова"`

#### `date`
- **Тип**: дата
- **Формат**: "YYYY-MM-DD"
- **Описание**: Дата написания отзыва
- **Пример**: `date = "2025-02-01"`

### Опциональные поля

#### `role`
- **Тип**: строка
- **Описание**: Должность или роль автора отзыва
- **Пример**: `role = "CEO компании"`, `role = "Частный предприниматель"`

#### `rating`
- **Тип**: число
- **Диапазон**: 1-5
- **Описание**: Оценка по пятибалльной шкале
- **Отображение**: Автоматически конвертируется в звездочки (⭐⭐⭐⭐⭐)
- **Пример**: `rating = 5`

#### `weight`
- **Тип**: число
- **Описание**: Вес для сортировки отзывов
- **Сортировка**: Меньшие значения отображаются первыми
- **По умолчанию**: Сортировка по дате
- **Пример**: `weight = 1`

#### `image`
- **Тип**: строка
- **Описание**: Путь к фотографии клиента
- **Формат**: Относительный путь от корня сайта
- **Пример**: `image = "/img/anna.jpg"`

## Управление отзывами

### Редактирование существующего отзыва
1. Откройте файл отзыва в `content/reviews/`
2. Внесите изменения в метаданные или текст
3. Сохраните файл
4. Hugo автоматически обновит отображение на всех страницах

### Удаление отзыва
1. Удалите файл отзыва из `content/reviews/`
2. Отзыв автоматически исчезнет со всех страниц

### Временное скрытие
Для временного скрытия отзыва можно:
1. Переименовать файл, добавив префикс `_` (например, `_hidden-review.md`)
2. Или переместить в подпапку `content/reviews/drafts/`

## Отображение отзывов

### Автоматическое подключение
Отзывы автоматически отображаются на всех страницах благодаря подключению в основных шаблонах:

```go-html-template
{{ partial "reviews" . }}
```

### Шаблон отображения
Отзывы рендерятся через partial `layouts/partials/reviews.html`, который:

1. **Собирает отзывы**: `{{ $reviews := where site.RegularPages "Section" "reviews" }}`
2. **Сортирует по весу**: `{{ $reviews = sort $reviews "Params.weight" "asc" }}`
3. **Отображает в сетке**: Адаптивная сетка с карточками

### Структура карточки отзыва
Каждый отзыв отображается в виде карточки со следующими элементами:

- **Текст отзыва**: Основной контент
- **Автор**: Имя автора (жирным шрифтом)
- **Должность**: Должность/роль (серым цветом)
- **Рейтинг**: Звездочки в соответствии с оценкой
- **Дата**: Форматированная дата

## Стилизация отзывов

### CSS классы
Система использует следующие CSS классы для стилизации:

```css
.reviews-grid          # Контейнер для всех отзывов
.review-card          # Карточка отдельного отзыва
.review-text          # Текст отзыва
.review-author        # Имя автора
.review-position      # Должность автора
.review-rating        # Рейтинг (звездочки)
.review-date          # Дата отзыва
.reviews-title        # Заголовок секции отзывов
```

### Кастомизация стилей
Для изменения внешнего вида отзывов отредактируйте файл `assets/css/custom.css`:

```css
.review-card {
  background: #f8f9fa;           /* Цвет фона */
  padding: 25px;                 /* Внутренние отступы */
  border-radius: 10px;           /* Скругление углов */
  border-left: 4px solid #007bff; /* Левая граница */
}
```
