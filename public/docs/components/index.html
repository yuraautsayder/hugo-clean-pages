Компоненты и шаблоны

В этом разделе описаны все созданные компоненты, шаблоны и их настройки для проекта Hugo.

Partials (включаемые фрагменты)

partials/head.html

Назначение: Метаданные страницы, подключение стилей и SEO-оптимизация.

Функциональность:

  • Динамическое определение типа страницы (Home, Single, List, About)
  • Подключение кастомных CSS стилей
  • Мета-теги для поисковых систем

Использование:

{{ partial "head" (dict "Kind" "Home" "Template" "Home") }}
{{ partial "head" (dict "Kind" .Kind "Template" "Single") }}

partials/header.html

Назначение: Шапка сайта с навигационным меню.

Функциональность:

  • Адаптивное навигационное меню
  • Динамическое определение активной страницы
  • Поддержка многоуровневой навигации

Настройка меню: Меню настраивается в hugo.toml:

[menus]
  [[menus.main]]
    name = 'Главная'
    url = '/'
    weight = 10
  [[menus.main]]
    name = 'О компании'
    url = '/about'
    weight = 20
  [[menus.main]]
    name = 'Docs'
    url = '/docs'
    weight = 30

partials/footer.html

Назначение: Подвал сайта с дополнительной информацией.

Функциональность:

  • Контактная информация
  • Ссылки на социальные сети
  • Копирайт и правовая информация

partials/reviews.html

Назначение: Система отображения отзывов клиентов.

Функциональность:

  • Автоматический сбор всех отзывов из секции reviews
  • Сортировка по весу (weight) для управления порядком
  • Отображение рейтинга звездочками
  • Адаптивная сетка отзывов

Особенности:

  • Отзывы автоматически отображаются на всех страницах
  • Поддержка дополнительных полей: position, rating, weight
  • Автоматическое форматирование даты

Шаблоны страниц

Главная страница: layouts/index.html

Назначение: Главная страница сайта со списком всех статей.

Функциональность:

  • Отображение всех статей из секции posts
  • Карточки статей с изображениями, заголовками и датами
  • Адаптивная сетка с поддержкой мобильных устройств

Структура карточки статьи:

  • Изображение (если указано в image)
  • Заголовок с ссылкой
  • Краткое описание (Summary)
  • Дата публикации

Одиночная страница: layouts/_default/single.html

Назначение: Шаблон для отображения отдельных статей и страниц.

Функциональность:

  • Заголовок статьи
  • Дата публикации
  • Автор (если указан в author)
  • Полный текст статьи
  • Автоматическое отображение отзывов

Поддерживаемые поля:

+++
title = "Заголовок"
date = "2025-02-01"
author = "Имя автора"  # опционально
+++

Список страниц: layouts/_default/list.html

Назначение: Шаблон для отображения списков страниц (например, архив статей).

Функциональность:

  • Заголовок секции
  • Описание секции
  • Список всех страниц с карточками
  • Единообразный стиль с главной страницей

Страница “О компании”: layouts/_default/about.html

Назначение: Специальный шаблон для страницы “О компании”.

Функциональность:

  • Заголовок компании
  • Обложка (если указана в image)
  • Основной контент
  • Автоматическое отображение отзывов

Активация: Укажите в content/about.md:

+++
title = "О компании"
layout = "about"
image = "/img/company.jpg"  # опционально
+++

Стили и CSS

assets/css/custom.css

Назначение: Кастомные стили для всех компонентов сайта.

Основные стили:

Карточки статей

.posts-grid {
  display: flex;
  gap: 20px;
}

.thumb img {
  width: 100%;
  max-height: 300px;
  object-fit: cover;
}

Система отзывов

.reviews-grid {
  display: flex;
  flex-direction: column;
  gap: 20px;
  margin-top: 30px;
}

.review-card {
  background: #f8f9fa;
  padding: 25px;
  border-radius: 10px;
  border-left: 4px solid #007bff;
}

Адаптивность

@media (max-width: 964px) {
  .posts-grid {
    flex-wrap: wrap;
  }
  .posts-grid > div {
    width: 100%;
  }
}

Подключение компонентов

В шаблонах

{{ partial "head" (dict "Kind" .Kind "Template" "Single") }}
{{ partial "header" (dict "Kind" .Kind "Template" "Single") }}
{{ partial "reviews" . }}
{{ partial "footer" . }}

Передача параметров

Компоненты поддерживают передачу параметров через словарь:

{{ partial "head" (dict "Kind" "Home" "Template" "Home") }}

Кастомизация

Добавление новых полей

Для добавления новых полей в карточки статей или отзывов:

  1. Добавьте поле в фронтматтер контента
  2. Обновите соответствующий шаблон
  3. При необходимости добавьте стили в CSS

Изменение стилей

Все стили централизованы в assets/css/custom.css. Для изменения дизайна:

  1. Отредактируйте CSS файл
  2. Перезапустите Hugo сервер для применения изменений

Создание новых partials

Для создания новых компонентов:

  1. Создайте файл в layouts/partials/
  2. Подключите через {{ partial "имя" . }}
  3. Добавьте необходимые стили
